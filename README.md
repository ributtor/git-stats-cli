# git-stats-cli

Visualize git repository statistics from the command line.

## Features
- Commit frequency charts
- Top contributors ranking
- File change heatmap
- Branch comparison
- Code churn analysis

## Usage
```bash
# Overview stats
git-stats summary

# Top contributors
git-stats contributors --limit 10

# Commit activity
git-stats activity --period weekly

# File hotspots
git-stats hotspots --limit 20
```
